# RakshEx User Segmentation

> Who uses RakshEx and why.
> Date: 2026-05-17

---

## SEGMENTS

### Segment 1: Security-First (25%)

**Profile:**

- Role: Security engineer, DevOps, CTO
- Team: 20-200 engineers
- Pain: Can't see what AI agents are doing in production
- Trigger: Security audit, compliance requirement

**Behavior:**

- Imports all collections immediately
- Checks every finding manually
- Runs scans before every deploy
- Shares reports with team

**Wow moment:** Finding a leaked API key their team forgot about
**Churn risk:** False positives, slow scans
**Retention driver:** Real security incidents prevented

**Target:** Enterprise upsell, SOC2 sales

---

### Segment 2: Cost-Optimizer (30%)

**Profile:**

- Role: Engineering manager, founder
- Team: 2-10 engineers
- Pain: Surprise $5K+ monthly API bills
- Trigger: Budget overrun, investor pressure on burn

**Behavior:**

- Checks cost dashboard first
- Sets up budget alerts immediately
- Explores every cost-related feature
- Wants to see ROI in dollars

**Wow moment:** Discovering hidden reasoning costs are 3× visible costs
**Churn risk:** Costs don't decrease, false confidence
**Retention driver:** Measurable money saved

**Target:** Pro plan, team plan

---

### Segment 3: AI Builder (35%)

**Profile:**

- Role: AI/ML engineer, indie hacker, technical founder
- Team: 1-5 people
- Pain: AI agents going rogue, infinite loops, unexpected costs
- Trigger: First major AI agent deployment

**Behavior:**

- Explores AgentGuard first
- Tries demo mode extensively
- Tests with edge cases
- Provides detailed feedback

**Wow moment:** AgentGuard stopping an infinite loop that would have cost $200
**Churn risk:** Feature not mature enough, too many false stops
**Retention driver:** AgentGuard actually preventing disasters

**Target:** Pro plan, referral program

---

### Segment 4: Curious Explorer (10%)

**Profile:**

- Role: Developer, student, hobbyist
- Team: 1 person
- Pain: Wants to learn about AI security
- Trigger: Saw on Twitter/Product Hunt, curiosity

**Behavior:**

- Plays with demo
- Imports one collection
- Runs a few scans
- Doesn't return for days

**Wow moment:** Seeing how many security issues exist in AI code
**Churn risk:** High — not solving urgent problem
**Retention driver:** Finding real issues in their own code

**Target:** Free tier, content marketing
