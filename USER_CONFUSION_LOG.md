## HOW TO USE THIS LOG

1. **Every onboarding call:** Add at least 2 entries.
2. **Every support ticket:** Add 1 entry.
3. **Every user interview:** Ask "What was confusing?"
4. **Review weekly:** Sort by severity and user count.
5. **Fix top 3 every week.** No exceptions.
