## EMERGENCY PROTOCOLS

### If negative comment appears

1. Reply within 5 minutes
2. Acknowledge the issue honestly
3. Offer to fix it or explain why it's designed that way
4. Take detailed note for product improvement
5. Never argue or get defensive

### If technical issue occurs

1. Pin a comment: "We're investigating X"
2. Fix within 1 hour if possible
3. Update comment when fixed
4. Thank users for patience

### If falling behind in rankings

1. Post on additional channels (Reddit, Discord, Telegram)
2. Ask beta users for one more push
3. Post update highlighting new features
4. Engage more aggressively in comments
