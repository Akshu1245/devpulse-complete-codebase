# RakshEx Community Building Playbook

> Turning users into advocates through transparency and belonging.
> Date: 2026-05-17

---

## COMMUNITY CHANNELS

### 1. Discord Server

**Server Structure:**

```
📢 Announcements (read-only)
💬 General
🆘 Help & Support
🐛 Bug Reports
💡 Feature Requests
🏆 Show & Tell
📊 Beta Feedback
🤝 Introductions
```

**Engagement Rules:**

- Founder responds to every message within 4 hours (during IST working hours)
- Weekly "What's New" post every Monday
- Monthly AMA (Ask Me Anything) voice chat
- Beta users get special "Early Adopter" role

**Launch Message:**

```
👋 Welcome to the RakshEx community!

We're building the security layer for AI-powered applications.

• Share what you're building
• Report bugs (we fix within 24h)
• Request features (we build what users ask for)
• Help other developers

 founders are here daily. No corporate BS.

Discord invite: [link]
```

### 2. GitHub Discussions

**Categories:**

- Q&A — technical questions
- Ideas — feature requests with voting
- Show and tell — what users built

**First 5 Discussion Seeds:**

1. "What API security issues have you found with RakshEx?"
2. "What's your biggest hidden API cost?"

## ADVOCACY PROGRAM

### RakshEx Champions

**Criteria:**

- Active user for 30+ days
- Referred at least 1 user
- Contributed feedback or bug report
- Posted about RakshEx publicly

**Benefits:**

- Free Pro for life
- Early access to new features
- Monthly private AMA with founder
- Swag (stickers, t-shirt)
- Listed on "Champions" page

**Recognition:**

- Twitter shoutout
- Blog post interview
- Speaking opportunity at events
- Input on roadmap

## TRUST BUILDING TACTICS

### Radical Transparency

1. **Publish failures:** "We had a 2-hour outage. Here's what happened."
2. **Publish decisions:** "Why we chose X over Y"
3. **Publish metrics:** "Our FP rate was 25%. Here's how we fixed it."
4. **Publish roadmap:** Public Trello/GitHub project board
5. **Publish pricing rationale:** "Why we charge $29/mo"

### Founder Presence

- Reply to every Twitter mention
- Answer every Discord question personally
- Write every blog post (not ghostwritten)
- Share personal learnings and mistakes
- Be reachable: akshay@rakshex.in
